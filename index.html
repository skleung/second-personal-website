About

Front-end developer, jazz pianist, and researcher, I'm a sophomore at Stanford University majoring in CS. I'm interested in mobile health applications and building web applications.

I started learning front-end development three weeks ago, and have built three websites to date. I'm taking classes this quarter to pick up some more back-end knowledge this quarter so I can start building my first applications

Projects

I recently declared CS, so this page is somewhat sparse. Check back every once in a while to see what I've been up to.

The following projects I put together over winter break 2013:

Sight

Sight is a location-enabled application that allows users to open a site to find friends and coordinate events and meet-ups. It uses geolocation data from the Android API and the backend was built with PHP/MySQL.

This is my group's entry for Stanford's annual Hackathon hosted by Stanford ACM and Stanford BASES. I was responsible for the front-end and my buddies work on the backend using PHP/MySQL. Unfortunately, the application is still a work in progress and isn't fully functional yet. Some features I learned whiel building the application:

  • Twitter Bootstrap
  • Google Maps API
  • AJAX calls

Stanford Pre-med APAMSA

I'm part of a new premed society on campus, and I wanted to design a website to get our name out there. I used LESS, JQuery, and a bit of PHP to get the email list feature working. Some features I had in mind when I designed it:

  • Docking navbar
  • Interactive members section
  • Designing the logo for our chapter

Personal Website

My first foray into web design. I built a personal portfolio just to get some experience with HTML/CSS. I learned how to use cool features from JQuery and make things interactive. Some features I thought were cool:

  • Toggling sections/colors
  • Dynamic scrolling
  • Having a single index.html

Research

I'm currently an undergraduate researcher in the Chang laboratory under the mentorship of Dr. Robert Spitale and Ryan Flynn. My current projects involve characterizing new protein interactions that target non-coding RNA segments.

Before joining the Chang lab, I performed structural studies to further rational vaccine design for RSV and HIV-1 at the Vaccine Research Center under the direction of Dr. Jason Gorman and Dr. Jason McLellan.

Outside of biomedical research, I've been working on and off on an independent research project that models organic chemistry using music theory.
